# Release Notes

## Version 1.1.0 (Version Code: 2)

### What's New

**New Features:**
- Swipe left to delete items quickly
- Progress bar showing shopping completion
- Category color indicators for easy identification
- Collapsible input form for cleaner interface
- "Hide completed" toggle to focus on remaining items
- Improved empty state with helpful guidance

**Visual Improvements:**
- Strikethrough effect for completed items
- Smooth animations throughout the app
- Category colors in dropdown menus
- Material Design 3 styling
- Better status bar integration

**Under the Hood:**
- Improved data reliability
- Better error handling
- Optimized app size with R8
- Enhanced backup support
